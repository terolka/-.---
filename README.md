# Анализ поведения пользователей в мобильном приложении
## На основе данных использования мобильного приложения для продажи продуктов питания проанализировать воронку продаж, а также оценить результаты А/А/В тестирования
### Используемые инструменты и навыки: А/В тестирование, Python, Pandas, Matplotlib, Seaborn, событийная аналитика, продуктовые метрики, Plotly, проверка статистических гипотез, визуализация данных
